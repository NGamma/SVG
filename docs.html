SVG Reference

group

group()

creates a <group> element

like a layer in Photoshop, a group is a container. it's useful for z-ordering. it doesn't show up visually, but if you style it, the style will apply to all its children.

line

line(x1, y1, x2, y2)

creates a <line> element

this is a 2D line segment, defined by its endpoints (x1,y1) and (x2,y2).

circle

circle(x, y, radius)

creates a <circle> element

a circle is defined by its center (x,y) and its radius.

ellipse

ellipse(x, y, rx, ry)

creates a <ellipse> element

an ellipse is defined by its center (x,y) and its radius along two axes rx and ry. the axes of the ellipse are always aligned to the x,y axes, you have to apply a rotate transform to make it otherwise.

rect

rect(x, y, width, height)

creates a <rect> element

the (x,y) point defines the top left corner of the rectangle.

polygon

polygon(pointsArray)

creates a <polygon> element

a polygon is a closed shape defined by a series of points that trace the outline of the polygon. the stroke will automatically connect the last point back to the first.

polyline

polyline(pointsArray)

creates a <polyline> element

similar to a polygon, a polyline is defined by a sequence of points, but the stroked path will not connect the last point to the first.

bezier

bezier(fromX, fromY, c1X, c1Y, c2X, c2Y, toX, toY)

creates a <path> element

a bezier curve is defined between two points (fromX, fromY) and (toX, toY) with two control points (c1X, c1Y) and (c2X, c2Y). The curve is encoded as a path object.

arc

arc(x, y, radius, angleA, angleB)

creates a <path> element

an arc is the curve along the boundary of a circle. the arc is encoded as a path object.

wedge

wedge(x, y, radius, angleA, angleB)

creates a <path> element

a wedge is a filled arc, like a slice of a pie that includes the center of the circle. the wedge is encoded as a path object.

regularPolygon

regularPolygon(cX, cY, radius, sides)

creates a <polygon> element

this is a convenient function for creating regular polygons of n number of sides, centered at point (cX, cY), with a circumscribed radius.

text

text(textString, x, y)

creates a <text> element

the x and y describe the location of the left most point along the baseline. the text sits above the point in the y direction. if defined at (0,0) the text will sit above the top of the frame.
